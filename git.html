Introduction to Git


What is Git?

Git is a version control system that helps developers track every change they make to their projects. It keeps a history of what was done and lets you go back if needed. Git is free, openâ€‘source, and used by programmers all over the world.


Why Use Git?

Developers use Git because:

  • It saves all changes made to a project
  • It allows multiple people to work together
  • It helps you fix mistakes easily
  • It keeps a full history of your work so nothing is lost

Example : Writing a School Report with Friends

Imagine you and your friend are working together on a school report. You both make changes to the same document. Without Git, you might accidentally overwrite each otherâ€™s work or lose previous versions. Git acts like a history tracker: it remembers every change, who made it, and when. You can go back to any previous version if something goes wrong. Git also lets both of you work at the same time safely, and later combine your changes without losing anything.


In short, Git makes teamwork easier and helps you keep your work safe and organized!

Installing Git

To install Git on your computer, follow these steps:

  1. Go to the official Git website: https://git-scm.com/downloads
  2. Select your operating system (Windows, macOS, or Linux)
  3. Download the installer and run it
  4. Follow the setup instructions, keeping the default options unless you have specific needs
  5. After installation, open your command line (Terminal on macOS/Linux, Command Prompt or Git Bash on Windows) and type git --version to verify that Git is installed correctly.

Use this if you don't understand something in the guide, or if you're having trouble installing Git. You can always ask for help from a teacher or mentor.
